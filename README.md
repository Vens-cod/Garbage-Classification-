## 项目概述

系统使用 **ResNet50**（ImageNet 预训练权重）在 40 类生活垃圾数据集上微调，最终在 Gradio 中提供可视化垃圾分类识别服务。

---

## 功能特性

### 用户系统
- 用户注册与登录（密码 PBKDF2-SHA256 哈希加密）
- 管理员/普通用户双角色权限
- 个人信息查看与密码修改
- 管理员可管理用户（创建、删除、重置密码）

### 图像识别
- 上传图片进行垃圾分类识别
- 返回 Top-K 置信度结果
- 识别记录自动保存与历史查询（分页、搜索）
- 识别快照保存与详情查看
- 支持 CSV 导出识别记录

### 模型管理
- 上传 `.pth` 模型权重文件并自动注册
- 切换当前生效的推理模型
- 查看模型列表（架构、大小、精度、训练轮数等）

### 模型训练
- 内置 ResNet50 训练脚本，支持 GPU/CPU
- Gradio 页面中一键启动训练，实时流式显示训练日志与进度
- 训练完成后自动注册模型并设为当前模型
- 可配置超参数：学习率、批大小、权重衰减、训练轮数
- 支持 AlexNet、CNN 等架构扩展

## 环境配置

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 MySQL 数据库

创建数据库 `garbage`，导入表结构：

```bash
mysql -u root -p < garbage.sql
```

### 3. 配置环境变量（可选）

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `MYSQL_HOST` | MySQL 主机 | `根据情况更改` |
| `MYSQL_PORT` | MySQL 端口 | `3306` |
| `MYSQL_USER` | MySQL 用户 | `root` |
| `MYSQL_PASSWORD` | MySQL 密码 | `自己的` |
| `MYSQL_DATABASE` | 数据库名 | `garbage` |
| `DB_URL` | 完整数据库 URL（优先于上述变量） | — |
| `PORT` | Gradio 服务端口 | `6006` |

## 使用指南

### 登录页面
输入用户名和密码登录；管理员可进入模型管理与用户管理功能。

### 图像识别
1. 切换到「图像识别」选项卡
2. 上传垃圾图片
3. 选择推理权重（或使用当前模型）
4. 点击「开始识别」查看分类结果与置信度

### 识别记录
- 查看历史识别记录（支持分页与关键词搜索）
- 点击表格行查看详情和图片快照
- 一键导出 CSV

### 模型管理
- **上传模型**：上传 `.pth` 权重文件
- **设置当前模型**：从已有模型中选择推理使用的模型
- **训练模型**：选择架构、配置超参数、启动训练

## 数据集

### 类别信息

系统支持 **40 类**生活垃圾识别，分为四大类：

| 大类 | 数量 | 示例 |
|------|------|------|
| 其他垃圾 | 6 类 | 一次性快餐盒、污损塑料、烟蒂、牙签、竹筷等 |
| 厨余垃圾 | 8 类 | 剩饭剩菜、大骨头、水果果皮、茶叶渣、蛋壳等 |
| 可回收物 | 23 类 | 塑料瓶、玻璃杯、纸板箱、易拉罐、旧衣服等 |
| 有害垃圾 | 3 类 | 干电池、过期药物、软膏 |

### 数据划分
- 训练集：`data/train_list.txt`
- 验证集：`data/validate_list.txt`
- 测试集：`data/test_list.txt`

## 模型训练

### 命令行训练

```bash
python train/train_resnet50.py \
    --epochs 10 \
    --batch-size 32 \
    --lr 3e-4 \
    --weight-decay 1e-4 \
    --pretrained
```

### 训练参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--epochs` | 训练轮数 | 10 |
| `--batch-size` | 批大小 | 32 |
| `--lr` | 学习率 | 3e-4 |
| `--weight-decay` | 权重衰减 | 1e-4 |
| `--pretrained` | 使用 ImageNet 预训练权重 | 否 |
| `--cpu` | 强制使用 CPU | 否 |
| `--num-workers` | 数据加载线程数 | 2 |
| `--seed` | 随机种子 | 42 |
| `--out-dir` | 模型输出目录 | `models/` |
| `--run-name` | 运行名称 | 自动生成 |

### Gradio 页面训练
在「模型管理 → 训练模型面板」中配置参数并一键启动，训练日志实时输出，完成后自动注册模型。

## API 参考

### 数据库 (db.py)

`Database` 类封装了用户、记录、模型、设置四张表的 CRUD 操作，支持 MySQL 后端。

### 配置 (config.py)

`AppConfig` 类读取环境变量或默认值，提供数据库 URL 和端口配置。

### 训练脚本 (train/train_resnet50.py)

`train()` 函数完成数据加载、模型构建（ResNet50 + 自定义全连接层）、训练循环、验证评估与 Checkpoint 保存。
